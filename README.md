# BMW E60 Coder Pro

Full-featured Android app for BMW E60 diagnostics, coding, and tuning via K+DCAN.

Built with Expo + React Native.

## Features
- K+DCAN USB communication
- ECU Coding & Daten files
- Fault code reading/clearing
- Performance tuning
- F-Series steering wheel support
- Remote start
- Xbox controller drive mode (experimental)
